# tmux-museum

Reference tooling for studying the tmux source. `tmux-museum/src` is a
pristine copy of the tmux source tree from https://github.com/tmux/tmux.
It is never modified -- all build activity happens out-of-tree.

## Refresh

`tmux-museum/bin/refresh-museum.sh` rebuilds all generated artifacts:

    tmux-museum/bin/refresh-museum.sh          # everything
    tmux-museum/bin/refresh-museum.sh xref     # cscope + ctags
    tmux-museum/bin/refresh-museum.sh gdb      # debug build
    tmux-museum/bin/refresh-museum.sh pp       # preprocessed views
    tmux-museum/bin/refresh-museum.sh link     # refresh rebuild/ symlinks

Default `pp` output is a small set of hot files. Pass explicit sources
for more:

    tmux-museum/bin/refresh-museum.sh pp cmd-split-window.c grid.c

## Layout

- `src/` -- pristine tmux source (read-only, never modified)
- `build/src/` -- disposable copy where autotools runs (gitignored)
- `build/gdb/` -- debug build tree
- `out/xref/` -- cscope database and tags
- `out/gdb/` -- debug binary, symbols, build log
- `out/pp/` -- curated preprocessed `.i` files
- `rebuild/` -- symlinks into `out/` for editor navigation
- `bin/refresh-museum.sh` -- rebuilds the above

`build/`, `out/`, and `rebuild/` are gitignored and regenerated by the
script. Override paths with environment variables (`TMUX_ROOT`,
`OUT_ROOT`, `BUILD_ROOT`, `REBUILD_DIR`) if needed.

## Pristine source policy

`src/` is never modified by the build. The script copies it to
`build/src/` and runs autotools there. Cross-referencing (xref, ctags)
reads directly from the pristine tree.
